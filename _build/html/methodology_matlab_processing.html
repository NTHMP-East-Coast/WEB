Examples for making stations

  1. run a MATLAB script (sample: CRM_read_gridB_FL_for_GB.m) to generate grid B (make sure using MHW)

  2. run a MATLAB script (sample: gridC_Brevard.m) to generate grid C

  3. make stations for coupling grids A and B (sample: AB_mk_station_G_FL.m)

  4. make stations for coupling grids B and C (sample: BC_mk_station_PM_BR.m)
